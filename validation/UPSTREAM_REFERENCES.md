# Upstream reference files consulted

This crate was shaped against the public `pdfplumber` API and the behavior of these upstream modules:

- `pdfplumber/page.py`
- `pdfplumber/text.py`
- `pdfplumber/table.py`
- `pdfplumber/utils/geometry.py`
- `pdfplumber/utils/clustering.py`
- `tests/test_basics.py`
- `tests/test_table.py`
- `tests/test_dedupe_chars.py`

The fixture goldens in `tests/goldens/goldens.json` were generated by running `pdfplumber` against the bundled PDFs.
